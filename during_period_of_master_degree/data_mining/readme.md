
# 動機
- 研究顯示，學生選課會依未來的目標、以前的修課模式，去選擇關聯性高的課程進行學習，在不少研究也指出這些教師教學風格、課堂成績表現會影響到學生選課偏好
- 我們觀察xx大學學生的修課成績與選課資訊，希望能找出先前的修課成績與後來的修課表現間是有關聯的
